# MIDNIGHT GRIND — Game Design Document v3.0

See the full GDD in the project task description.

## Quick Reference

### Story Structure (Harmon Circle)
- **Act 1: The Come-Up** (0-50 hrs) — Arrive, discover, prove yourself
- **Act 2: The Rise** (50-125 hrs) — Empire building, path divergence
- **Act 3: The Fall** (125-150 hrs) — Lose everything
- **Act 4: The Redemption** (150-200 hrs) — Rebuild, confront Zero, determine legacy

### Characters
- **Maya Torres** — The Heart (mechanic, emotional anchor)
- **Diesel** — The Mentor (broken legend)
- **Nikko Cross** — The Rival (desperate, not arrogant)
- **Jade Winters** — The Temptation (corporate path)
- **Zero** — The Antagonist (unseen force, not a person)
- **Ghost** — The Legend (never speaks, 1% encounter chance)

### 6 Endings
1. Legend of the Streets
2. The Professional
3. Destroy Zero
4. Become Zero
5. The Escape
6. The Ghost (secret)

### REP Tiers
0. UNKNOWN (0-999)
1. NEWCOMER (1,000-4,999)
2. KNOWN (5,000-14,999)
3. RESPECTED (15,000-34,999)
4. FEARED (35,000-74,999)
5. LEGENDARY (75,000+)

### Design Pillars
1. Every Race Tells A Story
2. Authentic Ownership
3. Deep Mechanical Truth
4. Meaningful Stakes
5. Relationships Are Everything
6. Unified Challenge
